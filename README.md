Group players, then set ownership for the whole group at once from the standard Ownership Configuration dialog. Also adds a small shield icon on sidebar entries to show who owns a document.

## Usage

- **Settings → Configure Settings → Player Groups → Manage Groups** to create/edit groups.
- Right-click anything → **Configure Ownership**: group rows appear above the user list. Changing a group updates its members.
- Hover the shield icon on sidebar entries to see who owns it.
